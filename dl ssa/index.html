History

Facial emotion recognition originated with psychologist Paul Ekman's cross-cultural studies identifying six basic emotions expressible universally.[1] His 1971 paper demonstrated identical expressions among isolated tribes.

Computerized FER began 1990s using Facial Action Coding System (FACS) – 44 action units (AUs) mapping muscle movements to expressions. Early systems rule-based, 50-60% accuracy.

Deep learning era (2013): FER2013 challenge saw CNNs achieve 73% accuracy vs 71% traditional ML. Today transformer models push 80%+.

Scientific Foundation

FER rooted in evolutionary psychology – Darwin (1872) proposed facial expressions innate for survival signaling. Modern neuroscience confirms amygdala activation during emotion processing.

Technology Pipeline

  1. 1. Face Detection

    MTCNN/ViT-face → Bounding box → 48×48 crop

  2. 2. Preprocessing

    Grayscale, CLAHE contrast, normalization [0,1]

  3. 3. CNN Inference

    Conv2D(32/64/128) → GAP → Dense(128) → Softmax(7)

  4. 4. Post-processing

    Argmax + confidence → Emotion label
